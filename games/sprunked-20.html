Frequently Asked Questions

What's new in version 2.0?
▼
Version 2.0 introduces HD audio, real-time collaboration, cloud saving, enhanced visual effects, and many more features that elevate the music creation experience.
Can I import my creations from the original version?
▼
Yes! Sprunked 2.0 includes a project import feature that allows you to bring your original creations into the new version and enhance them with the new tools.
How does multiplayer work?
▼
The new multiplayer system allows real-time collaboration with other players. You can create music together, share project files, and even perform live collaborative sessions.
What are the system requirements?
▼
Sprunked 2.0 runs in any modern web browser with an internet connection. For the best experience, we recommend using an updated browser and a stable internet connection.
